# Academic Portfolio Website

A modern and responsive personal portfolio website built using HTML, CSS, and JavaScript. This portfolio showcases my academic background, technical skills, projects, and contact information in a clean and professional design.

---

## 🚀 Features

- Responsive modern UI design
- Animated skills progress bars
- Dark/Light mode toggle
- Typing text effect
- About Me section
- Education timeline
- Skills showcase
- Contact section
- Social media links
- Download CV button
